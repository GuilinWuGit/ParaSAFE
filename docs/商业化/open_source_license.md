# ParaSAFE 推荐开源协议

## 推荐协议
- **Apache License 2.0**：宽松、允许商用、专利友好，适合平台型项目。
- **MPL 2.0**：代码级开源，便于与闭源模块集成。
- **LGPL v3**：适合库/组件，允许与闭源系统链接。

## 选择建议
- 若希望最大化行业采纳和商业合作，推荐 Apache 2.0。
- 若需保护核心代码但允许商业集成，推荐 MPL 2.0 或 LGPL。

## 注意事项
- 明确版权归属和贡献者协议（CLA）。
- 商业客户如需闭源集成或特殊授权，可采用双许可证模式。
- 保持 LICENSE 文件和头部版权声明的规范性。
